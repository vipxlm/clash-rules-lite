粗野主义风格

风格定义

追求原始的HTML感，通过鲜明对比、单调字体、无修饰的元素，展现大胆、实验性甚至反主流的设计态度。

Eng Keywords: raw HTML feel, stark contrast, monospaced fonts, unadorned, bold, experimental, anti-establishment.

核心特征

  • 字体 主要应用 'Source Code Pro', 'Courier New', monospace。
  • 颜色 使用高对比度颜色。例如，亮黄色背景 (#FFFF00) 配黑色文字 (#000000)。避免柔和的颜色过渡和复杂的调色板。
  • 边框 元素（如卡片、按钮）可以有粗犷的、实色的边框 (例如 border: 3px solid #000000;)。
  • 元素 标签可以是简单的黑色背景、亮色文字，无圆角。避免圆角、平滑的阴影等修饰性效果。元素应显得直接、未加修饰。
  • 布局 布局可以简单直接，甚至略显“粗糙”。网格系统可以用于基本对齐，但不需要追求像素完美或复杂的布局结构。
